# API de Gerenciamento de Carteira de Investimentos
## Descrição

Este projeto é uma API RESTful desenvolvida em Java com Spring Boot para o gerenciamento de uma carteira de investimentos, permitindo cadastrar, listar, filtrar, atualizar e remover ativos, além de visualizar um resumo consolidado da carteira.

### Requisitos Técnicos Utilizados

* Java 17+ com Spring Boot 3.x
* Spring Data JPA com banco relacional: PostgreSQL

## Pré-requisitos para rodar o projeto
* Git
* Docker e Docker Compose instalados
* Java (JDK) 17

#### 1. Clonar o repositório
```

git@github.com:Ivynaaa/API-de-Gerenciamento-de-Carteira-de-Investimentos.git

```

#### 2. Inicie o banco de Dados
```

docker compose -f .\compose.yaml up  

```

Após esses passos inicie o backend e teste a aplicação utilizando o arquivo  `investment.rest`
